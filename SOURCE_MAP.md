# Source Map

本仓库是基于公开 GitHub 项目的重写展示版，不直接搬运原仓库代码。

## 参考仓库

1. `nanobrowser/nanobrowser`
   - License: Apache-2.0
   - 重点：浏览器内多 agent 自动化

2. `Azure/multi-agent-doc-research`
   - License: MIT
   - 重点：文档研究、多 agent 协作、报告生成

3. `tarun7r/deep-research-agent`
   - License: MIT
   - 重点：研究规划、检索、综合、写作、评分

4. `hupe1980/agentmesh`
   - License: Apache-2.0
   - 重点：多 agent 调度、状态管理、checkpoint、观测

## 我采用的处理方式

- 参考公开 README 里的问题定义和架构描述
- 提炼出统一的多 agent 闭环
- 重新写成一个独立的展示仓库
- 保留来源信息，避免把参考项目包装成原创源码
